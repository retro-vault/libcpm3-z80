libcpm3-z80

CP/M 3 Standard C Library for Z80
Maintained by retro-vault

Abstract

libcpm3-z80 is a minimal and readable C library targeting CP/M 3 on Z80 with SDCC. It provides core headers and implementations for common C runtime areas such as stdio, string, stdlib, math, ctype, and time, plus CP/M-specific support in sys/. This page documents build outputs, release artifacts, and link usage.

Overview

The repository is structured into:

  • src/: library implementation and CRT0 source.
  • include/: public headers to ship with the library.
  • docs/: this documentation site.
  • test/: test programs (not part of release artifacts).
Release artifacts for this project are intentionally limited to library consumers: libcpm3-z80.lib, crt0cpm3-z80.rel, and include/.

Build

Docker is the default build path. To build library artifacts only (without tests), run: 

make clean
make docker-lib

For a native build (requires SDCC toolchain installed), run: 

make clean
make lib

Artifacts

A successful library-only build produces:

Path Description
bin/libcpm3-z80.lib Static library archive.
bin/crt0cpm3-z80.rel CP/M 3 CRT0 object to link first.
bin/include/ Public headers copied from include/.

Use In Your Program

Link order matters on Z80/SDCC builds. Use CRT0 first, then your objects, then this library, then SDCC runtime stubs. 

sdcc --std-c11 --no-std-crt0 --nostdinc --nostdlib \
  -Ibin/include -c myprog.c

sdcc --std-c11 --no-std-crt0 --nostdinc --nostdlib \
  -o myprog.ihx \
  bin/crt0cpm3-z80.rel myprog.rel \
  bin/libcpm3-z80.lib lib/libsdcc-z80.lib
Practical rule: CRT0 must be first on the link line for correct startup and symbol resolution.

Provided Headers

The project ships standard C headers and CP/M specific system headers. Key examples:

  • stdio.h, stdlib.h, string.h, math.h, time.h.
  • ctype.h, errno.h, fcntl.h, unistd.h.
  • sys/bdos.h, sys/types.h, sys/stat.h.

Notes

The release pipeline publishes only library-consumer assets and does not package test programs. GitHub Pages publishing is independent from release publishing.
